System Maintenance

  • Advance System Care Free - Registry Cleanup, system optimiser and anti malware (old Version 3)
  • Avast - Antivirus
  • CCleaner - Registry cleanup
  • Desktop Icon Save and Restore - allows you to save the position of all your desktop icons etc and restore them if it gets messed up by a resolution change.
  • eicar - Test virus - completely harmless but will trigger a response from your Anti-Virus program
  • HijackThis - Check your registry entries for malware
  • MemTest - Windows memory test program
  • PenDriveLinux - how to make a bootable USB flash drive running Linux - I prefer to use Knoppix but the site covers various Linux flavours. The UNIX Grymoire will tell you more than you will ever need to know about Linux
  • PhotoRec - Photo (and other data) recovery from damaged disks including camera cards etc.
  • Recuva - Recover lost, damaged or deleted files.
  • Use this Portable version of Recuva from a USB drive if the data is on your hard drive as installing the main program may overwrite what you want to recover.
  • Speccy - Displays detailed system information - OS, CPU, Memory etc.
  • SyncBack Free - backup, restore or sync files between drives.
  • TreeSizeFree - Find out what's using up all your disk space (that links to an old version, current site here)
  • WinDirStat - Another prog which shows what's using up your disk space - this one gives a colour coded 'map' of the different file types
